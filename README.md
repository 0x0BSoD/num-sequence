## Numero uno


```sh
Usage:
        main {numbers} [Parameters]
Parameters:
        {numbers} Number for build array, required
        [-i | --itemSep] Separator for items in string, default ','
        [-s | --secSep] Separator for sequence in string, default '|'
Example:
        main '1|5,6,7|10'
        main '1%5.6.7%10' -i='.' -s='%'

```
